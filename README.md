# GYM PRO

אפליקציית אימון חכמה | Smart Workout Tracker

🔗 **Live:** https://t1ptip.github.io/gym-pro/

---

## מה זה?

GYM PRO היא אפליקציית Web (PWA) חד-קובצית לניהול תוכנית אימונים אישית.  
ניתן להתקין אותה על מסך הבית כמו אפליקציה נייטיב.

---

## פיצ'רים

- 📋 **תוכנית אימון חופשית** — הדבק טקסט בפורמט פשוט, האפליקציה מנתחת אוטומטית
- ✅ **מעקב תרגילים** — סמן תרגילים כ"הושלם" עם פס התקדמות
- ⚖️ **מעקב משקלים** — בחר משקל לכל תרגיל, נשמר אוטומטית
- 🔁 **סופר-סט** — תמיכה בשורת superset שמקבצת תרגילים בצהוב
- 📅 **שבועות כוח** — תמיכה בווריאנטים שונים לשבועות 1–3 מול שבוע 4/8
- 🌐 **דו-לשוני HE / EN** — טאגל מלא בין עברית לאנגלית (RTL / LTR), נשמר ב-localStorage
- ▶️ **יוטיוב** — קישור ישיר לחיפוש וידאו לכל תרגיל
- 🖼️ **תמונות** — קישור לחיפוש תמונות בגוגל לכל תרגיל
- 📲 **PWA** — Service Worker + Manifest להתקנה על מסך הבית
- 🔄 **איפוס יומי** — סימוני "הושלם" מתאפסים כל יום אוטומטית

## פורמט תוכנית אימון

```
אימון A — חזה + רגליים
אימון משלב כוח רגליים עם פלג גוף עליון
* בנץ' פרס: 4 סטים, 8-10 חזרות
* פלאי כבל: 3 סטים, 12-15 חזרות
* סקוואט: 4 סטים, 6 חזרות

אימון B — זרועות
סופר-סט לזרועות
סופר-סט
* כפיפת מרפקים: 3 סטים, 10-12 חזרות
* פשיטת מרפקים: 3 סטים, 10-12 חזרות
```

**כללים:**
- בלוקים מופרדים בשורה ריקה כפולה
- שורה 1 = שם הטאב
- שורה 2 = תיאור
- שורות 3+ = תרגילים (מתחילים ב-`*` או מספר)
- שורת `סופר-סט` / `superset` מתגת את התרגיל הבא לצהוב

---

## טכנולוגיה

- **Vanilla JS + HTML + CSS** — קובץ אחד, אפס תלויות
- **localStorage** — שמירת תוכנית, משקלים, מצב שבוע ושפה
- **PWA** — Service Worker + Web App Manifest
- **RTL/LTR** — תמיכה מלאה בכיוון עברית/אנגלית
